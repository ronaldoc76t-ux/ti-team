# Équipe TI - Projet

Repository pour l'équipe TI avec les profils d'agents et leur mémoire.

## Rôles - Architecture TI

| Rôle | Description |
|------|-------------|
| **Enterprise Architect** | Vision globale, alignement business-IT, référentiels |
| **Solution Architect** | Conception de solutions pour projets spécifiques |
| **Technical Architect** | Standards techniques, Stack technologique |
| **Data Architect** | Modèles de données, governance, analytics |
| **Security Architect** | Sécurité, conformité, gestion des risques |
| **Cloud Architect** | Stratégie cloud, infrastructure AWS/GCP/Azure |
| **Software Architect** | Architecture applicative, patterns, qualité |
| **Infrastructure Architect** | Serveurs, réseau, stockage, supervision |

## Rôles - Développeurs

| Rôle | Description |

| **Backend** | APIs, services, logique métier |
